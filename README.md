# Библиотечная база данных

Проект включает базу данных для библиотеки и консольное приложение на C# для работы с ней.

## База данных (MySQL/MariaDB)

- **8 таблиц**: author, genre, publishing, books, a_copy_of_book, readers, staff, book_distribution
- **3 триггера**: проверка дат, обновление статуса, защита от удаления автора
- **3 хранимые процедуры**: выдача книги, возврат книги, поиск по автору
- **Связи**: все таблицы нормализованы до 3НФ

## Консольное приложение (C#)

Реализовано 10 запросов:

1. Список авторов по алфавиту
2. Жанры в обратном порядке
3. Первые 5 книг по названию
4. Количество книг в библиотеке
5. Уникальные года издания
6. Самая старая книга
7. Поиск книги по ID
8. Количество доступных экземпляров
9. Годы регистрации читателей
10. Книга с наибольшим количеством экземпляров

## Технологии

- C# .NET 8.0
- MySQL / MariaDB
- MySql.Data

## Запуск

1. Импортируй `sql/schema.sql` в MySQL
2. Установи NuGet пакет: `Install-Package MySql.Data`
3. Настрой строку подключения в `Program.cs`
4. Запусти консольное приложение
